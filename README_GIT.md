# bECCsh - 纯Bash椭圆曲线密码学实现

## 🎯 项目简介

**世界首个纯Bash椭圆曲线密码学实现** - 完全零外部依赖的教育研究工具。

### ✨ 核心成就
- ✅ **完全零外部依赖** - 仅使用Bash内置命令
- ✅ **世界首创** - 第一个纯Bash椭圆曲线密码学实现  
- ✅ **极高教育价值** - 完美的密码学教学工具
- ✅ **技术突破** - 证明了Bash语言的极限能力

## 🚀 快速开始

### 体验纯Bash版本
```bash
# 体验核心纯Bash实现
cd core && ./becc_pure.sh

# 快速演示
./demo/quick_demo.sh

# 功能测试
./demo/pure_bash_tests/test_all_functions.sh
```

## 🛡️ 重要安全警告

⚠️ **仅适用于教育研究目的**  
⚠️ **不适合生产环境使用**  
⚠️ **使用简化算法，安全强度有限**  
⚠️ **随机数为伪随机，非密码学强度**  

### 技术限制
- 整数大小受Bash 32/64位限制
- 性能相对较低（纯解释执行）
- 算法实现为教育简化版
- 无法处理密码学级别的大数
