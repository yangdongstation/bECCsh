# bECCsh 专业技术展示页面

本项目新增了两个专业技术展示页面，旨在帮助用户深入理解椭圆曲线密码学的数学原理和技术实现。

## 📋 页面概览

### 🔐 [index_cryptographic.html](index_cryptographic.html)
**密码学技术详解页面** - 交互式技术展示

#### 内容特色：
- ✅ **完整算法流程** - ECDSA签名和验证的每个步骤
- ✅ **代码示例展示** - 纯Bash实现的实际代码
- ✅ **交互式演示** - 步骤化的算法执行过程
- ✅ **技术实现细节** - 从数学到代码的完整映射
- ✅ **安全特性说明** - RFC 6979、侧信道防护等

#### 适合人群：
- 🎯 想要理解技术实现细节的开发者
- 🎯 需要查看代码示例的工程师
- 🎯 对密码学算法实现感兴趣的学习者
- 🎯 想要了解零依赖实现的技术爱好者

---

### 📐 [index_mathematical.html](index_mathematical.html)
**数学原理展示页面** - 数学公式详细推导

#### 内容特色：
- ✅ **完整数学推导** - 从基础方程到算法证明
- ✅ **公式详细说明** - 每个符号和参数的含义
- ✅ **算法正确性证明** - ECDSA验证的数学证明
- ✅ **安全性分析** - 离散对数问题的数学基础
- ✅ **步骤化推导** - 循序渐进的数学推理

#### 适合人群：
- 📚 密码学专业的学生和教师
- 📚 数学和计算机科学的研究者
- 📚 想要深入理解算法原理的技术人员
- 📚 对椭圆曲线数学感兴趣的学习者
