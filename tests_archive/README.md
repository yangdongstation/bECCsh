
## 📋 各目录详细说明

### 🔧 core/ - 核心功能测试
存放核心功能测试脚本，包括综合测试和基础功能验证。

### 🔄 elliptic_curves/ - 椭圆曲线相关测试
包含椭圆曲线点运算、密钥生成、曲线参数验证等测试。

### ✍️ ecdsa/ - ECDSA签名验证测试
专门存放ECDSA签名生成和验证相关的测试脚本。

### 🔍 openssl_comparison/ - OpenSSL兼容性对比测试
与OpenSSL进行对比测试的脚本，验证兼容性。

### 🌡️ extreme_tests/ - 极限边界测试
测试边界条件、极端情况和错误处理的脚本。

### 🐛 debug_tools/ - 调试工具脚本
用于调试和问题诊断的工具脚本。

### 🎪 demos/ - 演示脚本
功能展示和演示用的脚本。

### 🎯 final_versions/ - 最终版本测试
项目最终阶段的验收测试和验证脚本。

### 🌍 environmental/ - 环境相关测试
环境检查、依赖验证等相关测试。

### 🧮 math_modules/ - 数学模块测试
基础数学运算、大数计算等数学相关测试。
