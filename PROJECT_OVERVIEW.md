# bECCsh 项目概览

## 🎯 项目定位

## 📁 目录结构

### 🔴 核心纯Bash实现（重点）
```
core/
├── becc_pure.sh              # 主程序 - 完全零依赖
├── examples/
│   └── pure_bash_demo.sh    # 纯Bash演示
├── lib/pure_bash/           # 纯Bash模块库
└── docs/                    # 技术文档
```

### 🟡 演示和测试（功能验证）
```
demo/
├── quick_demo.sh            # 快速演示入口
├── pure_bash_tests/         # 功能测试
├── validation/              # 验证测试
└── README.md               # 测试说明
```

### 🟢 历史归档（完整保留）
```
archive/                     # 开发历史归档
beccsh/                      # 完整原始实现
becc.sh, demo.sh            # 原始文件（兼容性）
```

## 🚀 快速开始

### 体验纯Bash版本
```bash
cd core
./becc_pure.sh
```

### 查看演示
```bash
./demo/quick_demo.sh
```

### 运行测试
```bash
./demo/pure_bash_tests/test_all_functions.sh
```
